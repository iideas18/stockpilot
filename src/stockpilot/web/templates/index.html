使用方式

1. 先用搜索找到股票，系统会把它放进全局工作区。

2. 在 分析工作台 查看 K 线、信号、形态，并把标的发送到回测 / Agent / 组合。

3. 在 对比实验室 对比 2-4 个标的的收益曲线与信号。

4. 在 回测工作室 保存多次实验并叠加历史结果。

5. 在 组合工作室 基于候选池做资产配置。
